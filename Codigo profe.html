Taller Práctico de HTML

¡Aprende HTML mediante ejercicios prácticos!

Introducción

En este taller aprenderás los conceptos básicos de HTML mediante ejercicios prácticos. Sigue las instrucciones de cada sección y completa los ejercicios propuestos.

Consejo: Abre un editor de texto como Visual Studio Code, Sublime Text o incluso el Bloc de notas para practicar.

Ejercicio 1: Estructura básica de HTML

Crea un archivo HTML con la estructura básica:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi primera página</title>
</head>
<body>
    <h1>¡Hola mundo!</h1>
    <p>Esta es mi primera página web.</p>
</body>
</html>

Tu turno:

Crea un archivo llamado mi_pagina.html y escribe el código anterior. Luego ábrelo en tu navegador para ver el resultado.

Ejercicio 2: Encabezados y párrafos

Los encabezados se definen con las etiquetas <h1> a <h6> y los párrafos con <p>.

Tu turno:

Agrega a tu página los siguientes elementos:

  • Un encabezado principal con <h1>
  • Un subtítulo con <h2>
  • Varios párrafos con <p>

Ejercicio 3: Enlaces e imágenes

Los enlaces se crean con la etiqueta <a> y las imágenes con <img>.

<a href="https://www.ejemplo.com">Visita nuestro sitio</a>

<img src="imagen.jpg" alt="Descripción de la imagen">

Tu turno:

Agrega a tu página:

  • Un enlace a tu página web favorita
  • Una imagen desde internet (usa la URL de la imagen)
  • Un enlace que abra en una nueva pestaña (investiga cómo hacerlo)

Ejercicio 4: Listas

Hay dos tipos de listas: ordenadas (<ol>) y no ordenadas (<ul>).

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>

<ol>
    <li>Primer paso</li>
    <li>Segundo paso</li>
</ol>

Tu turno:

Crea en tu página:

  • Una lista no ordenada con tus pasatiempos favoritos
  • Una lista ordenada con los pasos para preparar tu comida favorita

Ejercicio 5: Tablas

Las tablas se crean con las etiquetas <table>, <tr> (fila), <th> (encabezado) y <td> (celda).

<table>
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
    <tr>
        <td>María</td>
        <td>25</td>
    </tr>
</table>

Tu turno:

Crea una tabla con información sobre tus amigos o familiares con las columnas: Nombre, Edad y Ciudad.

Ejercicio 6: Formularios

Los formularios se crean con la etiqueta <form> y contienen diferentes tipos de inputs.

<form>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre"><br>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br>
    
    <input type="submit" value="Enviar">
</form>

Tu turno:

Crea un formulario de contacto que incluya:

  • Campo para nombre
  • Campo para email
  • Campo para mensaje (usando <textarea>)
  • Botón de enviar

Reto Final: Crea tu propia página web

Ahora que has practicado los conceptos básicos, crea una página web sobre tu tema favorito.

Requisitos:

  • Usa al menos 3 niveles de encabezados diferentes
  • Incluye varios párrafos de texto
  • Agrega al menos 2 imágenes
  • Incluye enlaces a páginas relacionadas
  • Crea una lista con información relevante
  • Diseña una tabla con datos interesantes
  • Agrega un formulario de contacto
